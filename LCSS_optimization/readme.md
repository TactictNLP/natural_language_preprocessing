1. longest common subsequence  
    最长公共子序列算法 是对有序队列 计算相似度时的常用算法
    原始的基于动态规划的方法，可能存在 对换两个sequence的顺序时，计算的公共子序列不同；
    而对于一些场景，又要求子序列的任意点不能是孤立的，尤其在NLP领域，孤立字很难体现公共含义；
    基于以上两点，对LCSS进行了改进
